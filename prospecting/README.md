## ICP 定义 (Ideal Customer Profile)

### Tier 1 客户 (P0)
- **大型食品进口商/生产商:** 年冷链支出 500 万 + RMB
- **医药/生物制品企业:** 年冷链支出 300 万 + RMB

### Tier 2 客户 (P1)
- **生鲜电商/社区团购平台:** 年冷链支出 200 万 + RMB
- **跨国餐饮连锁:** 年冷链支出 150 万 + RMB

### Tier 3 客户 (P2)
- **区域超市/零售商:** 年冷链支出 50-150 万 RMB
- **跨境电商 (生鲜类目):** 年冷链支出 50-100 万 RMB

## 目标客户行业分类

1. **食品进口商/生产商** (10 家)
   - 肉类进口商 (澳洲牛肉/猪肉)
   - 海鲜进口商 (三文鱼/龙虾)
   - 乳制品进口商 (奶粉/奶酪)
   - 冷冻食品生产商 (速冻水饺/火锅食材)
   - 水果进口商 (车厘子/榴莲)

2. **医药/生物制品** (8 家)
   - 疫苗/生物制品企业
   - 创新药企业 (生物药/抗体)
   - 医疗器械进口商 (诊断试剂)
   - 跨境电商药店

3. **生鲜零售/电商** (5 家)
   - 生鲜电商平台
   - 社区团购平台
   - 高端超市连锁
   - 餐饮供应链公司

4. **餐饮连锁** (8 家)
   - 火锅连锁 (50 家以上门店)
   - 西式快餐连锁

5. **超市/零售商** (4 家)
   - 区域超市连锁
   - 高端超市

## CRM 数据导入

### 数据库位置
**macOS:** `~/Library/Application Support/solo-crm/crm.db`

### 验证数据
```bash
sqlite3 ~/Library/Application\ Support/solo-crm/crm.db "SELECT COUNT(*) FROM companies;"
```

### 导出数据
```bash
sqlite3 ~/Library/Application\ Support/solo-crm/crm.db << 'EOF'
.mode csv
.headers on
.output companies_export.csv
SELECT * FROM companies ORDER BY notes LIKE '[P0]%' DESC;
.quit
EOF
```
