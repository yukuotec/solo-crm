## 使用说明

### 在 solo-crm 中查看数据

1. 启动应用：`npm run dev`
2. 点击 **Companies** 查看所有导入的公司
3. 点击 **Deals** 查看销售机会，按阶段筛选
4. 公司备注中包含 [P0]/[P1] 标识和预估支出

### 导出数据

```bash
sqlite3 ~/Library/Application\ Support/solo-crm/crm.db << 'EOF'
.mode csv
.headers on
.output companies_export.csv
SELECT * FROM companies ORDER BY notes LIKE '[P0]%' DESC;
.output deals_export.csv
SELECT * FROM deals ORDER BY stage, value DESC;
.quit
EOF
```
