## ⚠️ 注意事项

1. **信息时效性：** 联系人信息可能随时变动，建议定期核实
2. **隐私保护：** 仅限内部销售团队使用，不得外泄
3. **数据更新：** 建议每季度更新一次联系人信息
4. **公司匹配：** 导入脚本会自动匹配公司名，但需确保公司记录已存在

## 🔧 使用流程

### 方法一：使用 Node.js 脚本（推荐）

```bash
# 确保 CRM 应用已运行过（创建数据库）
node scripts/import-contacts.js
```

### 方法二：直接执行 SQL

```bash
# 先导入主要联系人
sqlite3 ~/Library/Application\ Support/solo-crm/crm.db < data/contacts_import.sql

# 再导入补充联系人
sqlite3 ~/Library/Application\ Support/solo-crm/crm.db < data/contacts_additional.sql
```
